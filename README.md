## Tóm tắt nhanh

- **D1:** ClawFriend định vị = Shares Trading + Skill Market + Social trên BNB; khác ClawHub (economic layer), Virtuals (skill store), GPT Store (Web3), Clawrr (shares/skill).
- **D2:** 8 skills (Whale Alert, Rug Check, Yield, Token Sniper, Smart Money, Airdrop Hunter, Portfolio, Gas) — mỗi skill có demand evidence và đề xuất public/private holder-gated.
- **D3:** $4.5K Twitter Ads, $3K KOL, $2K Event; $0 Blog + Community; partnership OpenClaw, BNB, DEX; timeline 4 tuần; target 300–700 sign-ups, 80–200 skill installs.
